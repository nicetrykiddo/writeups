# Writeups

This is the source code for my blog, hosted at [nicetrykiddo.github.io/writeups](https://nicetrykiddo.github.io/writeups).

It uses the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) Jekyll theme and is automatically deployed via GitHub Actions.

## Content

The content for this blog is generated from Obsidian markdown files in the private repository.
